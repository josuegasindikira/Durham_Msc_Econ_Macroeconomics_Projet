# Advanced Macroeconomics – DSGE Model in Dynare

This project implements and analyses a stochastic macroeconomic model in **Dynare/MATLAB**.

The model includes three AR(1) shocks:

- Total Factor Productivity (TFP) shock
- Investment-Specific Technology (IST) shock
- Preference shock

The project covers:

- Euler equation derivation
- Steady-state analysis
- Dynare implementation
- Implicit real wage and rental rate of capital
- Theoretical moments and correlations
- Variance decomposition
- Impulse Response Functions (IRFs)
